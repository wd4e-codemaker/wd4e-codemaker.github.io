    There are three types of lists in html.
  1. Ordered list (like this one)
  2. Unordered list(like the one provided to show default values of lists )
  3. Definition list(like the one used to provide definitions of these lists)
Ordered list :
The one having a finite sequence.uses "li" and "/li" for listing items code.

Ex: 1.,2.,3.,...;a),b),c),...;etc

Unordered list:
Doesn't have a finite sequence. Uses bullets.uses "li" and "/li" for listing items in code.

Ex:disc, hollow circle, arrow,etc

Definition list:
Used to write definitions . Uses "dt" for Definition title and "dd" for Definition data.

    Default values for different lists:

  • Ordered list: 1.,2.,3.,.....
  • Unordered list: disc
  • Definition list:Indentation for definition data
